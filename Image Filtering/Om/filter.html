Image Filtering

Filtering recalculates each pixel after operating it with a mask or kernal. It is used for blurring, sharpening, edge detection and many more purposes. There are two component while convoluting:

  • Image
  • Kernal - matrix of size nxn (where n is always odd). The sum of the whole matrix is needed to be zero or near zero. To perform masking the kernal matrix is convoluted over the image.

Convolution



In mathematics convolution is defined as the sum of the product of the two functions after one is reversed and shifted. However, since in masking reversing a function doesn't make a differece therefore simple product of each element is sumed to save computation.

Implementation in OpenCV Python

The method cv2::filter2d is used to apply any custom image filter.

dst = cv.filter2D(src, ddepth, kernel [, dst[, anchor[, delta[, borderType]]]]	)

Parameters

src, ddepth and kernal are essential parameter
src :input image.

ddepth :desired depth of the destination image, see combinations.

kernel :convolution kernel (or rather a correlation kernel), a single-channel floating point matrix; if you want to apply different kernels to different channels, split the image into separate color planes using split and process them individually.

anchor :anchor of the kernel that indicates the relative position of a filtered point within the kernel; the anchor should lie within the kernel; default value (-1,-1) means that the anchor is at the kernel center.

delta :optional value added to the filtered pixels before storing them in dst.

borderType :pixel extrapolation method, see BorderTypes. BORDER_WRAP is not supported.
